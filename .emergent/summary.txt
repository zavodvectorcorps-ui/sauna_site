<analysis>**original_problem_statement:**
Создай современный адаптивный сайт для польского производителя и продавца деревянных саун под ключ. Основная цель сайта — мотивировать клиента рассчитать цену и заказать сауну. Стиль — минимализм, светлое дерево, белый фон, чёрные и золотистые акценты.

**PRODUCT REQUIREMENTS:**
- Название компании: WM-Sauna
- **Структура сайта:**
  - **Главная страница (Hero-секция):** Большое фото, заголовок “Producent Saun Drewnianych w Polsce”, кнопки “Oblicz cenę” и “Zobacz ofertę”.
  - **Блок Модели саун (New):** Перед калькулятором должен быть блок с карточками моделей, которые можно выбирать в админке. Каждая карточка должна содержать галерею фото, описание и две кнопки: Сконфигурируй эту модель (открывает калькулятор с выбранной моделью) и Оставить заявку (открывает форму для заказа стандартной модели).
  - **Калькулятор саун:** Интерактивный модуль для расчета стоимости. Интеграция с существующим калькулятором через API (www.wm-kalkulator.pl).
  - **“Nasze realizacje” (галерея):** Сетка изображений с возможностью загружать свои фото и выборочно показывать фото из API.
  - **“Sauny dostępne od ręki”:** Управляемые через админку карточки моделей на складе (полный CRUD).
  - **Отзывы, О компании, Контакты:** Стандартные разделы.
- **Админка:** Полноценная панель для управления всем контентом сайта:
  - Редактирование текстов и заголовков всех разделов (PL/EN).
  - Управление изображениями в галерее (свои + из API).
  - Полное управление блоком В наличии (CRUD).
  - Управление новым блоком Модели саун.
  - Настройка действий для всех основных кнопок (якоря, ссылки, формы).
  - Управление оформлением (например, отступы между блоками).
  - Управление порядком секций.
